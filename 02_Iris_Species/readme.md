# 🌸 Iris Species Classification

## 📖 项目简介

这是一个基于经典 **Iris（鸢尾花）数据集** 的多分类小项目。  
目标是根据花萼和花瓣的特征，对鸢尾花的品种进行分类。  

这个项目适合作为机器学习入门练习，用来熟悉完整的分类任务流程 🚀

---

## 🧩 数据集信息

Iris 数据集共有：

- 🌼 **150 条样本**
- 📏 **4 个特征**
- 🏷 **3 个类别**

分类目标包括：

- `Iris-setosa`
- `Iris-versicolor`
- `Iris-virginica`

---

## 🎯 项目目标

本项目主要完成：

- 📂 加载 Iris 数据集
- ✂️ 划分训练集和测试集
- 🌲 使用随机森林进行建模
- 🔍 完成预测与分类评估

---

## 🛠 使用模型

当前项目使用：

- 🌲 **Random Forest Classifier**

随机森林适合作为入门分类任务的 baseline，简单稳定，效果也不错 ✅
