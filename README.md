# Machine Learning & Statistical Modeling Portfolio 🚀

## 📌 仓库简介 | Overview

本仓库用于系统整理我在**机器学习**与**统计学习**方向的学习、实践与复现工作，内容主要包括：

- **经典数据集与 Kaggle 项目实践**：从数据探索、特征工程到模型训练与结果分析；
- **统计学习算法复现**：对 EM、HMM、AdaBoost、GBDT、SVM 等经典算法进行原理推导与代码实现；
- **理论到工程的衔接**：结合数学理解、代码注释与实验结果，加深对模型机制的认识。

我希望通过这个仓库，持续沉淀自己在**统计学习基础、机器学习建模**以及**算法实现能力**方面的积累。
